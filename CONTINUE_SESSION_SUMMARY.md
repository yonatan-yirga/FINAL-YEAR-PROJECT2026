## 💡 Usage Patterns Established

### Icon Sizes
```jsx
// Empty states
<GraduationCap size={64} strokeWidth={1.5} />

// Section titles
<ClipboardList size={20} />

// Buttons
<Download size={18} />

// Inline with text
<MapPin size={16} />
```

### Integration Pattern
```jsx
// Import at top
import { ClipboardList, MessageSquare, Building2 } from 'lucide-react';

// Use in section titles
<h3 className="ai-card-title">
  <ClipboardList size={20} style={{ 
    display: 'inline-block', 
    verticalAlign: 'middle', 
    marginRight: 8 
  }} />
  Section Title
</h3>

// Use in buttons
<button>
  <Download size={18} style={{ marginRight: 8 }} />
  Button Text
</button>
```
