# Department Pages Redesign Plan
## Modern, Awesome & Premium Style

### Design System

#### Color Palette
- **Primary Blue**: `#3b82f6` → `#2563eb`
- **Success Green**: `#10b981` → `#059669`
- **Purple**: `#8b5cf6` → `#7c3aed`
- **Pink**: `#ec4899` → `#db2777`
- **Amber**: `#f59e0b` → `#d97706`
- **Gray Scale**: `#f8fafc`, `#f1f5f9`, `#e2e8f0`, `#cbd5e1`, `#94a3b8`, `#64748b`

#### Typography
- **Headers**: 24-32px, font-weight: 700-800
- **Subheaders**: 18-20px, font-weight: 600-700
- **Body**: 14-16px, font-weight: 400-500
- **Small**: 12-13px, font-weight: 500-600

#### Components Style
1. **Cards**: 
   - Border radius: 16-20px
   - Box shadow: `0 4px 20px rgba(0,0,0,0.06)`
   - Hover: `translateY(-4px)` + enhanced shadow
   
2. **Buttons**:
   - Border radius: 12px
   - Padding: 12px 24px
   - Gradient backgrounds
   - Smooth transitions (0.3s cubic-bezier)
   
3. **Tables**:
   - Rounded corners
   - Hover row effects
   - Gradient headers
   - Clean borders
   
4. **Filters**:
   - Pill-shaped buttons
   - Active state with gradient
   - Count badges
   
5. **Search Boxes**:
   - Icon inside input
   - Rounded (12px)
   - Subtle shadow
