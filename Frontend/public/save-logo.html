🎓 Debre Markos University Logo Setup

Follow these steps to add the official DMU logo to your application

📋 Instructions

  1. Right-click on the Debre Markos University logo image you have
  2. Select "Save Image As..."
  3. Save it with the filename: dmu-logo.png
  4. Save it to this location:
Frontend/public/dmu-logo.png

Expected Logo Preview

The official DMU seal with yellow border, landscape, and open book

Place your
DMU logo here
⚠️ Important: Make sure the filename is exactly dmu-logo.png (all lowercase) and it's placed in the Frontend/public/ directory.

✅ After Saving

  1. Refresh your application in the browser
  2. The DMU logo should appear in the header on all pages
  3. If you see "DMU" text instead, check the file path and name
