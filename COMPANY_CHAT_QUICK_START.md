# Company Chat Feature - Quick Start Guide

## 🎯 What Was Added

A **messaging system** that allows companies to communicate with their assigned students and advisors in a **three-way group chat**.

---

## 🚀 How to Access (Company Users)

### Option 1: From Dashboard
1. Login to your company account
2. Go to Dashboard (`/company/dashboard`)
3. Click the **"Messages"** card in the Management section

### Option 2: Direct Link
- Navigate to: `http://localhost:5173/company/messages`

### Option 3: Quick Links Sidebar
- Click **"Messages"** in the sidebar Quick Links

---

## 💬 What You'll See

### Conversations List (Left Side)
- All your active internships
- Student name + Advisor name
- Internship title
- Unread message count
- Last message preview

### Chat Window (Right Side)
- Full conversation history
- Messages from student, advisor, and you
- Send new messages
- Edit/delete your own messages

---

## 👥 Who Can See Messages?

Each conversation includes **3 participants**:

1. **Student** - The intern
2. **Advisor** - University supervisor  
3. **Company** - You (the employer)

**Everyone sees all messages** - it's a group chat!

---

## ✨ Key Features

✅ **Real-time messaging** - Instant delivery  
✅ **Notifications** - Get notified of new messages  
✅ **Read receipts** - See when messages are read  
✅ **Message history** - All past messages saved  
✅ **Edit messages** - Fix typos in your messages  
✅ **Delete messages** - Remove messages you sent

## 📱 Example Use Cases

### 1. Welcome New Intern
```
Company: "Welcome to TechCorp! Excited to have you on the team."
Student: "Thank you! Looking forward to starting."
Advisor: "Great to see the collaboration. John, remember to submit your weekly reports."
```

### 2. Project Update
```
Student: "I've completed the login feature. Ready for review."
Company: "Excellent work! I'll review it this afternoon."
Advisor: "Make sure to document your code, John."
```

### 3. Issue Resolution
```
Student: "I'm having trouble accessing the development server."
Company: "I'll have IT reset your credentials. Should be fixed in an hour."
Advisor: "Let me know if you need any additional support."
```
